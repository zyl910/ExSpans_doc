IndexOfAnyExcept<T>(ExSpan<T>, T)

Searches for the first index of any value other than the specified value.

public static nint IndexOfAnyExcept<T>(this ExSpan<T> span, T value) where T : IEquatable<T>?

Parameters

span ExSpan<T>

The span to search.

value T

A value to avoid.

Returns

nint

The index in the span of the first occurrence of any value other than value. If all of the values are value, returns -1.
